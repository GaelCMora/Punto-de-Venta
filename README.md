# Punto de Venta PWA

Una Progressive Web App (PWA) completa para gestión de ventas, productos y gastos.

## Características

✅ **Punto de Venta**
- Interfaz intuitiva para realizar ventas
- Búsqueda rápida de productos
- Carrito de compras con control de cantidades
- Sistema de descuentos
- Múltiples métodos de pago (efectivo, tarjeta, transferencia)
- Cálculo automático de cambio

✅ **Gestión de Productos**
- Agregar, editar y eliminar productos
- Organización por categorías
- Control de inventario/stock
- Código de producto único

✅ **Control de Gastos**
- Registro de gastos por categoría
- Seguimiento diario, semanal y mensual
- Notas y detalles de cada gasto

✅ **Reportes y Estadísticas**
- Ventas totales
- Gastos totales
- Ganancia neta
- Productos más vendidos
- Filtros por fecha

✅ **PWA**
- Funciona offline
- Instalable en móvil y escritorio
- Responsive design
- Tema claro/oscuro
- Almacenamiento local

## Instalación

1. Abre el archivo `index.html` en tu navegador
2. Para instalar como PWA:
   - **En móvil**: Toca el menú del navegador y selecciona "Agregar a pantalla de inicio"
   - **En escritorio**: Haz clic en el ícono de instalación en la barra de direcciones

## Uso

### Realizar una Venta
1. Ve a la pestaña "Venta"
2. Busca o selecciona productos
3. Ajusta cantidades en el carrito
4. Aplica descuento si es necesario
5. Haz clic en "Cobrar"
6. Selecciona método de pago
7. Completa la venta

### Agregar Productos
1. Ve a la pestaña "Productos"
2. Haz clic en "+ Nuevo Producto"
3. Completa la información
4. Guarda

### Registrar Gastos
1. Ve a la pestaña "Gastos"
2. Haz clic en "+ Nuevo Gasto"
3. Completa la información
4. Guarda

### Ver Reportes
1. Ve a la pestaña "Reportes"
2. Selecciona rango de fechas
3. Revisa estadísticas y productos más vendidos
